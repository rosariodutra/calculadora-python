# 🧮 Calculadora

Calculadora no terminal desenvolvida em Python com operações básicas, avançadas, conversor de unidades e histórico exportável.

## 📸 Preview

```

  Cálculos realizados: 3

  1. Operações Básicas
  2. Operações Avançadas
  3. Conversor de Unidades
  4. Ver Histórico
  0. Sair
```

## ✨ Funcionalidades

- **Operações básicas:** adição, subtração, multiplicação, divisão, módulo e potência
- **Operações avançadas:** raiz quadrada/cúbica, logaritmos, trigonometria, fatorial e valor absoluto
- **Conversor de unidades:** temperatura, distância e peso
- **Histórico da sessão** com horário de cada cálculo
- **Exportação do histórico** em arquivo `.txt`

## 🚀 Como executar

```bash
# Clone o repositório
git clone https://github.com/rosariodutra/calculadora-python.git

# Entre na pasta
cd calculadora-python

# Execute
python calculadora.py
```

> Requer Python 3.6+. Sem dependências externas.
